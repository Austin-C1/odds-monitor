# OddsMonitor V3

OddsMonitor V3 is a local tool for Polymarket monitoring, copy trading, position checks, and Telegram notifications.

## Features

- Multi-account management
- Copy-trading monitoring
- Position management
- Telegram notifications
- Large bet monitoring
- Market betting query
- GitHub online updates

## Run Locally

```powershell
.\launch-OddsMonitor.ps1
```

Default URL:

```text
http://127.0.0.1:18880
```

## Updates

Open the announcements page and click the update button in the top-right corner. Updates replace program files only and preserve local config and data.

Local config files are not uploaded to GitHub:

- `config/local.env.ps1`
- `config/update.json`

Example config:

- `config/update.example.json`
